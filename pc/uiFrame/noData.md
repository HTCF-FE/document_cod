# 暂无数据

用于向服务器发送ajax请求数据但没有数据的情况。

![PNG](./img/nodata.png) 

`DOM`结构：

```
<div class="loading_NoData">
	<<!-- 没有数据显示 -->
	<div class="noDataCon">
		<i class="iconfont">&#xe645;</i>
		<p class="secText">暂无相关数据</p>
	</div>
</div>
```

需要引入`noData.less`或`noData.css`。

由于每个区域的大小不同，需要显示该结构的位置也不同，因此这里不提供该结构的位置样式，需要在每个页面单独进行设置。
